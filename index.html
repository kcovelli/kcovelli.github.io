Bag of Crafting Helper

Enter the number of consumables you have available for crafting and everything you can make will appear! If you have a second monitor/play in windowed mode, you can hover over an input and scroll to change the amount without having to pause the game.

Unlike this tool by @eluc_, or this tool by @Luhof the recipes for each item are calculated on the fly instead of hard coded so (usually) more items are shown! Code adapted from bladecoding on github
